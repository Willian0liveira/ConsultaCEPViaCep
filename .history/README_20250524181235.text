📦 Consulta de CEP - Playground JS

Este é um projeto simples de consulta de CEPs utilizando JavaScript, HTML e a API pública do ViaCEP. O objetivo é permitir que o usuário informe um CEP e visualize informações relacionadas como cidade, estado, logradouro e bairro.

--------------------------------------------------

🚀 Funcionalidades

✅ Consulta de informações de endereço a partir de um CEP.  
✅ Tratamento de CEP inválido ou não encontrado.  
✅ Exibição de valores padrão quando o input estiver vazio.  
✅ Interface web simples e funcional.

--------------------------------------------------

🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com divisão modular dos estilos)
- JavaScript (ES6+)
- ViaCEP API (https://viacep.com.br)

🎯 Como funciona

1. O usuário insere um CEP no campo de texto.
2. Ao clicar no botão "Consultar", o sistema:
   - Se o input estiver vazio → exibe "- / -" como valor padrão.
   - Se o input estiver preenchido → faz uma requisição fetch para a API ViaCEP.
3. Os dados retornados são exibidos nas áreas correspondentes: Cidade, Estado, Logradouro e Bairro.
4. Caso o CEP seja inválido, são exibidas mensagens padrão de erro.

--------------------------------------------------

⚙️ Como usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Insira um CEP no campo de texto e clique no botão "Consultar".

--------------------------------------------------

❗ Observações

- A API ViaCEP não possui um campo específico para "região". Atualmente, o campo "Região" exibe "- / -".  
  (É possível implementar uma lógica adicional que associe o estado à sua respectiva região brasileira).

- O projeto está totalmente implementado com tecnologias de front-end, sem dependências externas.
